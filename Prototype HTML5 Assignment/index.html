Properly Creating the Content of an HTML5 Document

When you are creating the content for an HTML5 document, you must include specific information. The information has to be precise because it is what persuades the users to return to the site. Some of this information includes headers, paragraphs, and the proper use of semantics. However, you cannot properly include this information without the use of tags.

What are Tags?

Tags are short commands that are used in order to correctly assign text with specific roles. Each tag is opened by using angle brackets, and closed by using a forward slash enclosed in angle brackets. For instance, the table below includes different types of tags, the reasons for using them, and examples of opening and closing those tags.

At times, it is necessary to include images within your HTML5 document. In order to include images, you have to include, can you guess it? Tags! The tags that are used in order to input an image are "img src=", and "alt". Img src stands for "image source" and it is followed by the link to the page where the image is from.
